本SDKはウェブブラウザ上でLive2D Cubismを使用したアプリケーションを
作成するためのソフトウェア開発キットです。

Live2Dライブラリと、サンプルプロジェクトが含まれます。
開発に際しては以下の文章を必ずお読みください。

------------------------------
	動作環境
------------------------------
	プログラム言語 : JavaScript
	ライブラリ形式 : JavaScript
	グラフィック環境 : WebGL
	対応プラットフォーム : WebGL対応ブラウザ
	サンプルの開発環境 : JetBrains WebStorm


------------------------------
	描画について
------------------------------
	HTML5のcanvas要素を使ったWebGLで描画しています。
	WebGLをサポートしていないブラウザでは描画できません。
	また、デバイスの性能にも依存するため、実行環境によっては動作が重くなります。

------------------------------
	サンプルについて
------------------------------
	サンプルプロジェクトがsampleフォルダ以下に含まれます。

	Simple
		最も単純なLive2Dモデル組み込みのサンプルです。
		Live2Dに関する全ての基本処理を１ファイルに記述しています

	SampleApp1
		基本的な機能を実装したサンプルです。
		モーションの再生、表情の設定、ポーズの切り替え、物理演算の設定
		などを行います。
		右クリックでモデルを切り替えることができます

	SimipleMultiCanvas
		Simpleをベースにして、複数のキャンバスにモデルを表示しています。
